Subsets (Power Set)

Problem Statement

Given an integer array nums of unique elements, return all possible subsets (the power set). The solution set must not contain duplicate subsets. Return the solution in any order.

Example: Input: nums = [1,2,3] â†’ Output: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]

Approach 1: Backtracking (Recursive)

Explanation (Brute/Standard): This is the classic "choose/don't choose" backtracking approach. We define a recursive function that takes the current index. For each element, we make two decisions: either include the element in our current subset and recurse, or not include it and recurse. When we reach the end of the array, the `current` subset is a valid one to add to our results.

Time Complexity: O(N * 2^N) because there are 2^N subsets, and for each, we may need O(N) time to create a copy.

Space Complexity: O(N) for the recursion stack depth.

function subsets(nums):
    result = []
    backtrack(result, [], nums, 0)
    return result

function backtrack(result, current, nums, start_index):
    add copy of current to result
    
    for i from start_index to length(nums)-1:
        current.push(nums[i]) // Include
        backtrack(result, current, nums, i + 1)
        current.pop() // Backtrack (Don't include)

Approach 2: Cascading (Iterative)

Explanation (Better): Start with an empty list of subsets. Iterate through each number in the input array. For each number, iterate through the current list of subsets and create a new subset by adding the current number to each one. Add these new subsets to the main list. This effectively doubles the number of subsets at each step.

Time Complexity: O(N * 2^N)

Space Complexity: O(N * 2^N) to store the output.

function subsets(nums):
    result = [[]] // Start with the empty subset
    for num in nums:
        new_subsets = []
        for current_subset in result:
            new_subset = copy(current_subset)
            new_subset.push(num)
            new_subsets.push(new_subset)
        result.extend(new_subsets)
    return result

Approach 3: Bit Manipulation

Explanation (Optimal): For a set of N elements, there are 2^N possible subsets. We can map each subset to a unique integer from 0 to 2^N - 1. The binary representation of this integer acts as a bitmask. If the i-th bit of the integer is 1, it means the i-th element of the input array is included in the subset; otherwise, it's not. We can loop from 0 to 2^N - 1 and generate each subset accordingly.

Time Complexity: O(N * 2^N)

Space Complexity: O(N * 2^N) to store the output.

function subsets(nums):
    n = length(nums)
    result = []
    for i from 0 to (2^n - 1):
        subset = []
        for j from 0 to n-1:
            // Check if the j-th bit is set in i
            if (i >> j) & 1:
                subset.push(nums[j])
        result.push(subset)
    return result
